Password-Locking

Verilerinizi şifrelemek için geliştirilen bir paket
Nasıl kullanılır



Paketi yüklemek için

npm i password-locking

Paket kullanımı
const lang = require("password-locking");




Şifreleme

Tanımlanan ŞiFre 123456789 (-_-) (^-^) Yazısını anlamsız karakterlerile şifreler

const password = lang.convert("ŞiFre 123456789 (-_-) (^-^)")

Şifre çözme

Şifreleme kodun da kullandığımız password gelen veri şifreli veridir bunu anlaşılır hale getirmek için translation fonksiyonunu kullanırız

password verisi yerine şifreli kayıt edilmiş verileri yazıp deşifre edebilirsiniz

const tokenresult = lang.translation(password)



Örnek

const lang= require("password-locking");
const şifre = lang.convert("ŞiFre 123456789 (-_-) (^-^)");
const çözülmüş_şifre = lang.translation(şifre);
console.log(şifre);//Şifrelenmiş veri console.log(/r); console.log(çözülmüş_şifre);//"şifre" şifreli metini çözülmüş hali
