Password-Locking

An enhanced package to encrypt your data
How to use



To install the package

npm i password-locking

Package usage
const lang = require("password-locking");




Encryption

Defined Password 123456789 (-_-)(^-^) Encrypts text with nonsense characters

const password = lang.convert("Password 123456789 (-_-) (^-^)")

Decoding

The password we use in the encryption code is encrypted data, we use the translation function for clarity.

Instead of password data, you can write and decrypt encrypted stored data.

const tokenresult = lang.translation(password)
